== Example: reading JSON from Bugzilla

=== JSON Example: Bugzilla

https://wiki.mozilla.org/Bugzilla:REST_API[Bugzilla] plans to add a REST interface serving JSON.
You can just call a URI and retrieve JSON.

Here are some examples from Mozilla which already downported this API.:

. Bugzilla URIs
|===
|URI |Description

|https://bugzilla.mozilla.org/rest/bug/35
|Get info about Bug 35

|https://bugzilla.mozilla.org/rest/bug/707428/comment
|Get all comments for the bug 707428
|===

Please note that some URIs return a `JSONObject` object while others return a `JSONArray`.
